OpenSesame

Introduction

OpenSesame is a fast, easy, and secure way to create and retrieve passwords in GNOME based environments. It uses GNOME-Keyring to store passwords securely. OpenSesame uses apg (Automated Password Generator FIPS-181) to generate secure but pronouncable passwords. This makes passwords easier to remember the password if necessary.

Commonly used methods for password management that users resort to ultimately decrease security.
  • reuse the same password
  • create overly simplistic passwords
  • write the password down somewhere in plain text
With OpenSesame there is no...
  • need to enter a master password every time a password is unlocked
  • need for browser plugins, OpenSesame can paste a password in most any password field
  • configuration to fiddle with
